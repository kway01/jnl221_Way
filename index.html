Drones dropped in Pakistan, Pitch Interactive

This project uses data from the Bureau of Investigative Journalism, and uses a timeline and a bar chart as dataviz elements. The timeline displays drone strikes in Pakistan from 2004 until 2015 and it demonstrates the steep increase in drone strikes in recent years. Furthermore, the bar chart uses stick figures to exhibit the amount of people in Pakistan that have died as a result of drone strikes each month. The main message of this project is to elucidate the quantity of drone strikes in Pakistan in recent years. This project also attempts to highlight how these drone strikes tend to have a significant effect on civilians.

This project is highly effective at conveying the intended message through dataviz. Firstly, the timeline used lines projecting downwards to personify a drone strike, successfully mimicking each individual done strike. This allows the viewer to visualize each observation in the dataset as an actual drone strike, rather than just viewing a number on a screen. Secondly, the bar chart uses stick figures to show the amount of deaths as a result of drone strikes. This allows the reader to associate the data with actual people, once again humanizing the dataset.

Moreover, this data is definitely visceral, as Mona Chalabi would say, or in other words, creates powerful internal feelings within the viewer. This is executed by using lines projecting downwards to represent drone strikes, and stick figures to represent people. The color scheme in this project also adds to the emotional connection, as the colors black and red create a gloomy and tragic feel. It is hard for humans to conceptualize straight numbers, but this project turned the numbers into illustrations of drone strikes in Pakistan which made the dataviz visceral and gave viewers an emotional connection.
